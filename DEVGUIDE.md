# Development Guide

## Requirements

- [pnpm](https://pnpm.io)
- [zephyr](https://github.com/coot/zephyr)

## Install dependencies

```bash
pnpm install
```

## Build

```bash
pnpm build
```

## Lint

```bash
pnpm lint
```

## Format code

```bash
pnpm format
```

## Run tests

```bash
pnpm test
```
